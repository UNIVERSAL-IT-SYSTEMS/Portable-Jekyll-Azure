Portable Jekyll
==============

The portable version contains everything which is required to run Jekyll v2.4 on Azure Website:

* Ruby 2.0
* Ruby development Kit

##Usage Instructions##
See https://github.com/tapanila/Jekyll-Azure-Websites/
